# Diwali Sales Analysis

## Overview
This project analyzes Diwali sales data to uncover insights into customer behavior, product performance, and sales trends during the Diwali festival period.

## Dataset
The dataset contains sales transaction data from Diwali season, including customer demographics, product categories, purchase amounts, and transaction details.

### Raw Data
- **File**: `Raw_data/Diwali Sales Data.csv`
- Contains the original, uncleaned sales data

### Cleaned Data
- **File**: `Clean_data and Analysis/Diwali Sales Data.csv.xlsx`
- Processed and cleaned version of the raw data
- Ready for analysis and visualization

## Data Cleaning Process
The raw data underwent the following cleaning steps:
- Removal of duplicates
- Handling missing values
- Data type corrections
- Standardization of categorical variables
- Outlier treatment

## Analysis
The analysis includes:
- Sales performance by product category
- Customer segmentation and demographics
- Regional sales distribution
- Seasonal trends and patterns
- Key performance indicators (KPIs)

### Presentation
- **File**: `PPT/Diwali-Sales-Data-Analysis rohit[1].pptx`
- Comprehensive presentation of findings and insights
